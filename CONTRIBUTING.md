# Contributing

Thanks for improving Codex Everyday Guide.

## Content Rules

- Keep examples original and practical.
- Do not copy paragraphs, screenshots, diagrams, or prompt templates from third-party guides.
- Link to official OpenAI documentation for product behavior, limits, security, and pricing.
- Mark dynamic facts with a verification date.
- Avoid community group promotion in the navigation or footer.

## Local Build

```bash
npm run build
```

The site is generated as static HTML and can be opened directly from `index.html` or served with any static file server.
