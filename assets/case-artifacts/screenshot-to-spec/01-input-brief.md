# 07 设计截图转实现规格

## 中文

### 任务摘要
把目标截图和当前页面截图转成可执行改版规格，输出组件映射、状态表、风险项和截图验收方式。

### 可读取材料
- 目标截图只提炼密度、色彩、层级和交互状态，不搬运品牌、图片、文案或业务数据。
- 当前页面截图用于定位本站组件：导航、侧栏、卡片、表格、按钮、搜索和页脚。
- 验收视口固定为 1440x1000 与 390x900。

### 输入样例

```text
截图任务
目标：提炼深色后台风的布局密度和组件状态
当前页面：首页、案例页、使用规范页
禁止：复用目标截图中的品牌、业务数字、图片、文案
输出：规格、组件表、桌面和手机验收截图
```

## English

### Task Summary
Turn target and current screenshots into an actionable redesign spec with component mapping, state table, risks, and screenshot acceptance.

### Readable Materials
- Target screenshot is used only for density, color, hierarchy, and states; brand, images, copy, and business data are not carried over.
- Current page screenshots locate this site's components: nav, sidebar, cards, tables, buttons, search, and footer.
- Acceptance viewports are fixed at 1440x1000 and 390x900.

### Input Sample

```text
Screenshot task
Goal: extract dark dashboard layout density and component states
Current pages: home, recipe page, usage policy
Prohibited: reuse target screenshot brand, business numbers, images, or copy
Output: spec, component table, desktop and mobile acceptance screenshots
```
