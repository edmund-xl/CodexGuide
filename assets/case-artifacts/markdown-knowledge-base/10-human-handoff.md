# 05 Markdown 知识库重整 - 人工交接清单

## 中文

- 原文只移动和补元数据，不删除正文。
- 待补事项单独列出，不能伪装成已完成知识。
- 合并前由用户抽查至少 3 篇笔记。

## English

- Original text is moved and annotated, not deleted.
- Missing items are listed separately, not presented as completed knowledge.
- The user should spot-check at least three notes before merging.
