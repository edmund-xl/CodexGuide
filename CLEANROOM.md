# Clean-Room Notes

This project intentionally uses a clean-room content approach.

- `https://codexguide.ai/` is treated as a layout and navigation reference.
- Page copy, examples, diagrams, SVG assets, CSS, JavaScript, and case studies in this repository are original.
- The community/chat-group section from the reference site is intentionally omitted.
- Case studies use different scenarios from the reference site.
- Product facts should be checked against official OpenAI sources before publication.

If future contributors copy code, screenshots, text, or other substantial material from a third-party project, they must preserve the applicable license and attribution.
