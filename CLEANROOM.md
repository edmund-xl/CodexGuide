# Clean-Room Notes / 原创边界说明

This repository uses a clean-room content approach.

本仓库采用 clean-room 内容策略。

## Rules / 规则

- Public sites may be used as layout and navigation references only. / 公开网站只能作为布局和导航参考。
- Page copy, examples, diagrams, SVG assets, CSS, JavaScript, and recipes must be original. / 页面文案、示例、图示、SVG 资源、CSS、JavaScript 和案例必须原创。
- Third-party screenshots, prompts, case studies, and source code must not be copied. / 不复制第三方截图、提示词、案例和源码。
- Product facts must be checked against official OpenAI sources before publication. / 产品事实必须在发布前核对 OpenAI 官方资料。
- If future contributors reuse third-party material, they must preserve the applicable license and attribution. / 如果未来贡献者复用第三方材料，必须保留对应许可和署名。

## Evidence / 证据

The generated site includes bilingual content blocks, original SVG diagrams, original recipe scenarios, and an automated verification script.

生成站点包含双语内容块、原创 SVG 图示、原创案例场景和自动验证脚本。
